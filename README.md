# Liquid-velocity-detectio

##液體移動速度偵測的matlab程式

#核心程式模組介紹

1.標定與互動介面
+ 提供圖形化使用者介面 (GUI)，讓使用者能手動選取區域 (ROI) 與初始追蹤參考點。
+ 提供滑桿調整影片，進度無需事先剪輯。
+ 整合比例尺校正功能，透過拉線標定已知長度，計算像素與物理單位的轉換率。

2.追蹤引擎
+ Canny 邊緣偵測：系統首選方案，精確定位流體與背景的交界，同時搭配ROI限制偵測範圍。
+ 多層次退避機制 (Fallback)：若邊緣偵測失敗，會自動切換至區塊分析 (Blob) 或時間差分法 (Temporal Difference)，確保追蹤不中斷。
+ 位移約束：透過 limitMove 函數限制單幀最大位移，有效排除影像雜訊導致的錯誤跳轉。

3.物理數據運算
+ 計算相鄰幀之間的歐幾里得位移量。
+ 結合影像幀率 (FPS) 與標定比例計算物理流速。

4.動態視覺化展示
+ 以動畫方式同步呈現原始影像、追蹤點軌跡與 ROI 範圍。
+ 即時繪製流速變化曲線，方便研究者觀察流體動態特徵。

5.儲存對話框
+ 彈出作業系統標準的檔案儲存視窗。它允許使用者手動輸入檔名並選擇儲存路徑，同時支援自定義的副檔名過濾器（.csv 與 .mat），優化使用者操作體驗。
